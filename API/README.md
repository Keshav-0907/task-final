## 🛠️ Installation

1. **Clone the Repository**

```bash
git clone https://github.com/Keshav-0907/onFin-task-be
cd bangalore-area-be
```

2. **Install Dependencies**
```bash
npm i
```

3. **Setup Environment Variables**
```bash
cp .env.example .env
```

4. **Start the Development Server**
```bash
npm run dev
```
